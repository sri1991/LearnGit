Git Commands are:

git config --global user.name "[firstname lastname]"#To initialize the git account to local terminal with user name
git config --global user,email "[user mail]"#To initialize the git account to local terminal with user mail
git clone -b branch-name URL#To clone from a specific branch
git status#To check the modifications done in files
git add {filename}#To add the specific file to the git stage
git add .#To add all the changed files to the git stage
git commit -m "please put your comments"#To commit the stages to github
git push#To push the code to github
git checkout -b new_branch_name#To create a new branch
git push origin branch-name#To push the changes to github to specific branch
